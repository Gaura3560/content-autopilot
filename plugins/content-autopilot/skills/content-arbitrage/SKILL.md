# Content Arbitrage

Buy knowledge cheap in one market, sell it at a premium in another.

## When to Activate

- User says `/arbitrage` or `/content-arbitrage`
- User asks "what unique angle can I bring?"
- User wants content nobody else has

## Commands

### `/arbitrage` — Scan for arbitrage opportunities
### `/arbitrage {source_field} → {target_field}` — Specific transfer
### `/arbitrage overseas` — English → Japanese information gaps

## Arbitrage Types

| Type | Source | Target | Example |
|------|--------|--------|---------|
| Cross-niche | Marketing | Engineering | "Marketing funnels for developer tools" |
| Cross-language | English | Japanese | "This US trend hasn't hit Japan yet" |
| Cross-industry | Healthcare | Business | "Hospital efficiency → office productivity" |
| Cross-era | Historical | Modern | "Ancient strategy → modern AI tactics" |
| Cross-format | Academic | Popular | "Research paper → practical guide" |


## Quality Gate

- [ ] Arbitrage opportunities based on real information gaps
- [ ] Source knowledge verified as accurate before transfer
- [ ] Target audience would genuinely benefit (not forced connection)
- [ ] Competition assessment is honest
- [ ] Cultural/contextual adaptation considered for cross-language
