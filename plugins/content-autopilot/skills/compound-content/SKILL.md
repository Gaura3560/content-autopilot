# Compound Content

News depreciates. Frameworks appreciate. Build content that gets more valuable every day.

## When to Activate

- User says `/compound` or `/compound-content`
- User asks "how do I create evergreen content?"
- User wants content with long-term ROI

## Commands

### `/compound {topic}` — Design compound content
### `/compound audit` — Which existing content has compound potential?
### `/compound types` — Show compound content formats

## Compound Content Types

| Type | Appreciation Rate | Example |
|------|------------------|---------|
| Framework/Model | Very High | "The 4-Step AI Adoption Framework" |
| Definitive Guide | High | "The Complete Guide to note Monetization" |
| Curated Database | High | "100 AI Tools Ranked by Category" |
| Annual Report | Medium-High | "{Niche} State of the Industry 2026" |
| Template Library | High | "10 Email Templates for Content Creators" |
| Benchmark Data | Very High | "AI Content Performance Benchmarks" |
| Decision Matrix | High | "Which AI Tool Should You Use? Decision Tree" |

```
  Creation time: {hours}

```

## Quality Gate

- [ ] Content has clear appreciation mechanism
- [ ] Update cadence is planned (not set-and-forget)
- [ ] Citation potential is realistic
- [ ] Compound type matches the topic
- [ ] Long-term traffic projection based on similar content patterns
