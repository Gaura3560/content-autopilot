# X Spaces Script

Live audio is the highest-engagement format. Plan it like a pro.

## When to Activate

- User says `/spaces` or `/spaces {topic}`
- User asks "plan an X Spaces event"
- User wants to host live audio

## Commands

### `/spaces {topic}` — Full event plan
### `/spaces solo {topic}` — Solo monologue format
### `/spaces interview {guest}` — Interview format
### `/spaces panel {topic}` — Panel discussion format

## Workflow

```
X Spaces Plan: "{topic}"
============================================

Format: {solo / interview / panel}
Duration: {30-60 min}
Best time: {from best-times.json, engagement data}

--- Pre-Event ---
Promotion plan (1 week before):
  Day -7: Announcement tweet + schedule the Space
  Day -3: Teaser thread about what you'll cover
  Day -1: Reminder tweet + IG story

--- Event Script ---

[0:00-0:05] Opening
  "{greeting + topic intro + what listeners will learn}"
  Set rules: "Raise hand to ask questions, I'll bring you up"

[0:05-0:15] Main Topic 1
  Key point: "{point}"
  Discussion prompt: "{question for audience}"
  Data/story to share: "{specific anecdote or stat}"

[0:15-0:25] Main Topic 2
  ...

[0:25-0:35] Audience Q&A
  Prepared questions (in case no one asks):
    Q1: "{question}"
    Q2: "{question}"

[0:35-0:45] Key Takeaways
  "3 things to remember from today:..."

[0:45-0:50] CTA + Close
  "If this was valuable: {follow/note/paid content}"

--- Post-Event ---
1. Summary thread within 1 hour
2. note article: "Key takeaways from our Spaces on {topic}"
3. Quote highlights for IG/X posts
4. Invite attendees to next event

============================================
```

## Quality Gate

- [ ] Script has clear structure with time markers
- [ ] Audience interaction points planned
- [ ] Backup questions prepared for quiet audiences
- [ ] Post-event content conversion plan included
- [ ] Promotion timeline covers awareness building
