List all available skills in this plugin by running:
```bash
ls ${CLAUDE_PLUGIN_ROOT}/skills/
```
Then show the total count with `ls ${CLAUDE_PLUGIN_ROOT}/skills/ | wc -l` and explain that the user can invoke any skill by name (e.g. `/trend-scout`, `/content-analytics`).
