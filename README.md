# Content Autopilot

SNS content automation plugin for Claude Code — AGI Lab Skills Marketplace format.

Automate daily content creation for **note**, **X (Twitter)**, and **Instagram** with trend-driven topic selection, platform-native writing, bestseller title logic, brand-consistent visuals, **cross-platform DRM funnel strategy** (X/Instagram → note → monetization), **content analytics**, **repurposing**, **competitor analysis**, **series planning**, **A/B title testing**, **engagement templates**, **batch generation**, **SEO optimization**, **carousels**, **newsletters**, **launch campaigns**, **voice scripts**, and more — **129 skills** total.

## Capability Levels

| Level | Requirements | Features |
|-------|-------------|----------|
| **Level 1** | Claude Code + WebSearch | Text content only, image prompts for manual creation |
| **Level 2** | + gemini-image or fal.ai MCP | Text + auto-generated images |
| **Level 3** | + X API credentials | Text + images + X auto-posting |

### Level 1 (Everyone)

Works out of the box with any Claude Code installation. Generates text content for all platforms and provides image generation prompts for tools like Canva, Midjourney, or DALL-E.

### Level 2 (Image MCP)

Add `gemini-image` or `fal.ai` MCP server to your Claude Code config for automatic image generation with brand colors and platform-correct sizes.

### Level 3 (X API)

Set environment variables for X API auto-posting:

```bash
export X_API_KEY="your-key"
export X_API_SECRET="your-secret"
export X_ACCESS_TOKEN="your-token"
export X_ACCESS_SECRET="your-secret"
```

## Bestseller Title Logic

Content Writer applies a research-backed title generation system derived from analysis of 90+ bestsellers, Nature research papers, and 32,487 A/B tests:

- **9 title logic categories**: Paradox, Numbers, Unexpected Combination, Enemy, Ideal Self, Secrets, Simplicity, Coined Term, Question
- **Combined logics sell ~3x more** than single-logic titles
- **Genre-matched recommendations** for optimal performance
- **Saturated pattern avoidance** (e.g., "XX no 9-wari" is overdone)

## Funnel Strategy (DRM)

Content Autopilot supports a built-in **Direct Response Marketing (DRM) funnel** that turns your social media presence into a monetization pipeline:

```
X / Instagram (TOFU)          note free (MOFU)         note paid (BOFU)
┌──────────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  Reach & Attract  │────▶│  Trust & Educate  │────▶│  Convert & Earn  │
│  30% of insight   │     │  80% of insight   │     │  100% + bonus    │
│  Curiosity gap    │     │  Free articles    │     │  Paid articles   │
│  CTA → note       │     │  Lead magnet      │     │  Membership      │
└──────────────────┘     └──────────────────┘     └──────────────────┘
```

### How It Works

1. **Setup**: Run `/setup-profile` and enable funnel mode (Question 6)
2. **Design**: Run `/funnel-designer` to create content pillars and a content calendar
3. **Execute**: Run `/daily-autopilot` — funnel-aware content is generated automatically:
   - X/Instagram CTAs drive traffic to note
   - note free articles build trust and deliver lead magnets
   - note paid articles/membership monetize your audience
4. **Balance**: Daily autopilot shows your TOFU/MOFU/BOFU balance and recommends the optimal stage

### Ideal Funnel Balance

| Stage | Target | Purpose |
|-------|--------|---------|
| TOFU | 50% | Maximize reach and new followers |
| MOFU | 30% | Build trust and demonstrate expertise |
| BOFU | 20% | Drive revenue through paid content |

### Backward Compatible

Funnel features are **opt-in**. If `funnel.enabled` is `false` in your profile, all skills work exactly as before with independent per-platform CTAs.
